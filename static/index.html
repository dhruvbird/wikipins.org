Q. What do you get when you cross Wikipedia with Pinterest? [1]

A. Wikipins

Wikipins organizes wikipedia categories into pinterest style pins that let you visually browse wikipedia categories and articles and lets you spot linkages between articles on wikipedia.

Wikipins also provides some special meta-categories such as:

  • Popular Articles and
  • Recently Viewed

So go ahead and visually explore wikipedia!

Wikipins is made using node.js and is mostly written in javascript. Wikipins is open source software, and you can find the code on github.

Wikipins isn't associated with Wikipedia or the Wikimedia foundation even though it uses icons from there.
